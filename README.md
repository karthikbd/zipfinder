# package_build

Mono-repo containing two independent offline-geocoding Python packages.

| Package | Description | PyPI |
|---|---|---|
| **[address-finder](./address-finder/)** | Offline address parser (parses free-text addresses into structured fields) | [![PyPI](https://badge.fury.io/py/address-finder.svg)](https://pypi.org/project/address-finder/) |
| **[zipfinder](./zip-finder/)** | Offline postal-code / geocode lookup | [![PyPI](https://badge.fury.io/py/zipfinder.svg)](https://pypi.org/project/zipfinder/) |

See each sub-folder for its own README and usage documentation.
